# eCherha Telegram Bot

Цей бот автоматично перевіряє статус авто за номером на сайті [echerha.gov.ua](https://echerha.gov.ua) та надсилає сповіщення в Telegram.

## Як запустити

1. Завантаж цей проект на GitHub.
2. Створи Web Service на [Render.com](https://render.com).
3. Додай змінні середовища:
   - TELEGRAM_TOKEN
   - CHAT_ID
   - PLATE
4. Запусти сервіс — бот працюватиме 24/7.
